У цій програмі використовується Flask та psycopg2
1.Інсталяція в терміналі:
$ pip install Flask
$ pip install psycopg2
2.Далі потрібно запустити проект після чого зайти на:
http://127.0.0.1:5000/
3. На головній сторінці можна вибрати таблицю з якою можна взаємодіяти;
4. На наступній сторінці вводимо id елементу;
5. Тепер можга оновлювати, створювати, видаляти записи у таблиці.
